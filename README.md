## 📂 仓库内容指南
1. Comedy Scripts (幽默段子)
- 这里收录了我创作的脱口秀剧本及幽默短文。
- 代表作： 《家有“贤"妻》、《祖国已多年未亲近》等。
- 创作逻辑： 探索如何通过逻辑错位（Misdirection）和结构重组产生幽默感，对研究 AI 生成内容的“趣味性”和“拟人度”有很高参考价值。

2. Creative Nonfiction (纪实长文)
- 侧重于叙事逻辑的严密性，故事与现实情境的融合，以及情感的起伏。
- 适用场景： 可作为大模型在长文本理解、情感识别、逻辑一致性方面的测试案例。

3. Travelogue (游记系列)
- 记录世界各地的足迹。
- 深度观察： 非简单打卡，包含对当地文化、历史与社会的深度思考（如墨西哥城）。
- 语料价值： 丰富的地理文化实体名词及跨文化叙述风格。
